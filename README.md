# Learnova - Modern E-Learning Platform

Learnova is a highly scalable, production-ready, full-stack E-Learning platform tailored to accommodate three distinct roles: **Learners**, **Instructors**, and **Administrators**.

Built with an immersive UI and a robust relational data model, Learnova allows users to manage, distribute, and consume premium educational content flawlessly.

---

## 🚀 Tech Stack

### Frontend (`learnova-app`)
* **Framework**: React.js with Vite (TypeScript)
* **Styling**: Tailwind CSS & Modern UI components
* **Routing**: React Router DOM
* **State Management/API**: Context API & Axios
* **Icons & Animation**: Lucide React, Framer Motion

### Backend (`Learnova-backend`)
* **Runtime**: Node.js
* **Framework**: Express.js
* **Database**: PostgreSQL (pg, pg-pool)
* **Authentication**: JSON Web Tokens (JWT) & bcrypt
* **Mock Data**: `@faker-js/faker`

---

## ✨ Core Features

### 👨‍🎓 Learner Portal
- **Dashboard**: Track your total points, active courses, and system rank (Beginner, Explorer, Scholar, etc.).
- **Course Discovery**: Browse Free, Premium, or Private courses globally.
- **Unified Player**: Watch video lessons, read documents, and take quizzes inside a beautiful immersive course player.
- **Secure Enrollment**: Smooth purchase flows with mock secure checkout for premium content.

### 👩‍🏫 Instructor Panel
- **Course Management**: Complete control to create, edit, and publish your own courses.
- **Curriculum Builder**: Manage video/document lessons and configure structured order.
- **Evaluation**: Create dynamic quizzes with multiple options and assess learners.
- **Reporting**: Live analytics tracking your students' progress, enrollments, and quiz performances.

### 🛡️ Admin Dashboard
- **Platform Overview**: See global metrics such as total system revenue, aggregate user counts, and completion stats.
- **User Management**: Direct control to oversee all learners and instructors on the platform.
- **Course Administration**: System-wide administrative access to inspect, publish, or delete any course.

## 🏁 Getting Started

### Prerequisites
* Node.js v18+
* PostgreSQL running locally or remotely

### 1. Database Setup
Create your PostgreSQL database (e.g., `learnova_db`).
Navigate into the `Learnova-backend` directory and configure your `.env` file:
```env
PORT=5000
DB_HOST=localhost
DB_USER=postgres
DB_PASSWORD=your_password
DB_NAME=learnova_db
DB_PORT=5432
JWT_SECRET=super_secret_key
```

### 2. Run the Backend & Seed Data
```bash
cd Learnova-backend
npm install

# Seed the database with 300+ students, 10 instructors, and 50 courses!
node seed.js

# Start the server (runs on Port 5000)
npm run dev
```

### 3. Run the Frontend
```bash
cd learnova-app
npm install

# Start the React app (runs on Port 8080)
npm run dev
```

### 4. Default Logins
* **Admin**: `admin@gmail.com` / `admin`
* *Learners/Instructors*: To test other specific roles, use the seeded accounts generated by `node seed.js`. All seeded accounts use the default password: `password123`.
